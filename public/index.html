Sample Data API

Generate data for placeholders, testing, or database seeding.

All APIs support seed parameter to indicate when stable results are needed.

GET /image API

Description

This API searches online for images matching your keyword and redirects to a result. Results are cached to help provide more stable responses when using the seed parameter.

Example URL

/image?keyword=cat

Parameters

Result

Resolved Image URL

GET /json API

Description

This API returns structured data in JSON format with locale-aware values. When count is specified, returns an array of objects, otherwise returns a single object.

Example URL

/json?name=users&fields=id+name+title&count=10&locale=zh-HK

Parameters

Notes

  • Field Separators: Fields can be separated by comma, plus sign, or space, e.g. id,name or id+name or id name
  • Data Generation: Currently using Faker.js. Will be enhanced with AI-powered generation for more context-aware and realistic content.
  • AI Guidance: Use remark parameter to guide AI generation (coming soon)
