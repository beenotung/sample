Sample Data API

Generate data for placeholders, testing, or database seeding.

All APIs support seed parameter to indicate when stable results are needed.

Image API

/image?keyword=...

This API searches online for images matching your keyword and redirects to a result. Results are cached to help provide more stable responses when using the seed parameter.
